Política de Privacidade

A VIVA SOLUCOES E PERFORMANCE LTDA, inscrita no CNPJ sob o nº 61.413.631/00012-7, tem o compromisso de proteger a sua privacidade.

1. Coleta de Dados

Coletamos informações necessárias para a prestação de nossos serviços de tecnologia e automação, sempre em conformidade com a LGPD (Lei Geral de Proteção de Dados).

2. Uso de APIs e Integrações

Este domínio é utilizado para validação de integridade junto à Meta Platforms, Inc. Dados de usuários processados via API são utilizados exclusivamente para as finalidades contratadas pelos nossos clientes.

3. Segurança

Implementamos medidas técnicas e administrativas para proteger seus dados contra acessos não autorizados.

4. Contato

Para questões sobre privacidade, entre em contato através do e-mail: contato@vivacorretorarj.com.br
